# PokeStats

This is at the moment a very simple and basic website that fetches data from https://pokeapi.co/ and displays all the base stats of any pokemon you want to search for. If someone finds this and wants to make a contribution or add a feature feel free to use a pull request. 

I took inspiration into making this because my nephew hates having to google sometimes just for looking up the stats of whatever pokemon he is searching.

## Live SIte 

To use the webisite you can just click this link!
https://pokstats.netlify.app/

## Features up to now

- Search Pokemon by name
- Display Pokemon sprite
- Show Pokemon type(s)
- Show all base stats (HP, Attack, Defense, etc.)

## How to run locally

If you want to run locally use these commands for the backend: 
- cd backend
- python main.py

And then for the backend:
- cd frontend
- npm install
- npm run dev
